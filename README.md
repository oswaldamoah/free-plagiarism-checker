# Free Plagiarism Checker

Free (yeah, free) AI-assisted *web plagiarism* verification with document upload, source checks,
similarity scoring, and PDF report export.

**Link:** https://free-plagiarism-checker.vercel.app/

## Run Locally
Vite & React App with TanStack Routing.
```bash
npm install
npm run build
npm run dev
```

Then open the local URL printed by Vite, usually `http://localhost:5173`.

To test a production build locally:

```bash
npm run build
npm run preview
```
